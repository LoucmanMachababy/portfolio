Puissance 4

À propos de ce projet

Ce projet de Puissance 4 est une implémentation interactive du célèbre jeu de société, développée entièrement en JavaScript. L'objectif était de créer une version "module" jouable du jeu en faisant de la "POO".

Le jeu permet aux utilisateurs de jouer contre un adversaire humain en local.

Technologies utilisées

HTML CSS JavaScript

Caractéristiques

  • Mode deux joueurs sur le même écran
  • Animations pour la chute des jetons
  • Détection des victoires lorsque 4 jeton sont alignés
