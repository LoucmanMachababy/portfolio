MyCinema

À propos de ce projet

MyCinema est un projet de gestion de base de données d'un cinéma fictif développé durant ma formation à Epitech. L'objectif principal était de créer une application web permettant de gérer et d'explorer une base de données complète de films, de distributeur, de genre et de pouvoir ajouter des nouveaux membres.

Ce projet m'a permis de développer mes compétences en gestion de bases de données MySQL, en PHP côté backend, et en création d'interfaces utilisateur dynamiques et interactives.

Technologies utilisées

HTML CSS PHP MySQL

Caractéristiques

  • Gestion complète des films par genre, production et titre
  • Interface de recherche et de filtrage
  • Ajout de membre coté personnel
  • Gestion de base de donnée SQL
