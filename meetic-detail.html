My Meetic

Description du projet

Développement d'une application de rencontres en ligne inspirée de Meetic. La plateforme permet aux utilisateurs de créer des profils détaillés, de rechercher des partenaires potentiels selon différents critères.

Fonctionnalités clés

  • Création de profil et informations personnelles
  • Système d'inscription et de connexion sécurisé
  • Hachage de mot de passe
  • Recherche avancée avec filtres (âge, localisation, centres d'intérêt)
  • Interface responsive adaptée mobile et desktop

Technologies utilisées

PHP MySQL JavaScript AJAX MVC Architecture
